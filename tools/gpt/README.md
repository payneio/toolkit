# GPT Tools

AI and LLM interaction tools.

## Tools

### [gpt](gpt.md)

Interact with OpenAI's GPT models from the command line.

**Quick start:**
```bash
export OPENAI_API_KEY=sk-...
gpt "Explain quantum computing"
cat article.txt | gpt "Summarize this"
```

See [gpt.md](gpt.md) for full documentation.
