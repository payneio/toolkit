# Web Scraping Tools

Tools for scraping and converting web content.

## Tools

### [mdscraper](mdscraper.md)

Convert web pages to clean Markdown format.

**Quick start:**
```bash
mdscraper https://example.com > article.md
mdscraper https://blog.com/post | less
```

See [mdscraper.md](mdscraper.md) for full documentation.
