# Browser Tools

Browser automation tools.

## Tools

### [browser](browser.md)

AI-powered browser automation using natural language instructions. Execute complex browser tasks from the command line.

**Quick start:**
```bash
export OPENAI_API_KEY=sk-...
browser "Go to google.com and search for python tutorials"
browser "Fill out the contact form on example.com"
```

See [browser.md](browser.md) for full documentation.
